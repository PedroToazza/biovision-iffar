# Training

Use esta pasta para o codigo de treino da IA.

Pode entrar no Git:

- scripts de treino;
- codigo do modelo;
- codigo de dataset/dataloader;
- scripts de avaliacao;
- scripts de predicao;
- configuracoes pequenas;
- README com instrucoes de reproducao.

Nao deve entrar no Git:

- dataset de imagens;
- checkpoints;
- modelos treinados;
- logs grandes;
- ambiente virtual;
- arquivos de cache.

Recomendacao: subir o codigo necessario para reproduzir o treino, mas salvar pesos treinados e datasets fora do repositorio principal.
